# Примеры конфигураций

Эта папка содержит примеры конфигурационных файлов для различных серверов и окружений.

## Файлы

- `nginx.conf.example` - пример конфигурации Nginx
- `ssh-config.example` - пример SSH конфигурации

## Использование

1. Скопируйте нужный `.example` файл
2. Удалите суффикс `.example` 
3. Настройте под свои нужды
4. Не коммитьте файлы без `.example` в Git (они должны быть в `.gitignore`)
