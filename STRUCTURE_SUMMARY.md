## 🎨 Дизайн в цифрах

### Цветовая схема
- **Primary**: Синий градиент (логистика, надёжность)
- **Accent**: Яркий акцент для CTA и иконок
- **Background**: Светлая база для читаемости

### Типографика
- **H1**: 48-80px (Hero заголовки)
- **H2**: 36-64px (Секционные заголовки)
- **H3**: 20-24px (Карточки)
- **Body**: 15-20px (Основной текст)

### Отступы
- **Секции**: 96-128px
- **Карточки**: 32-40px
- **Элементы**: 24-32px

## 📈 KPI для отслеживания

### Конверсия
- **Form Submission Rate**: цель >3%
- **Click-through Rate на CTA**: цель >5%
- **Time to Conversion**: среднее время до заявки

### Вовлечённость
- **Bounce Rate**: цель <50%
- **Avg Session Duration**: цель >2 мин
- **Pages per Session**: цель >3

### Качество трафика
- **Organic Search**: % от общего трафика
- **Direct Traffic**: % повторных визитов
- **Referral**: % от партнёров

## 🎨 Визуальные паттерны

### Карточки
```css
- Rounded corners (border-radius: 16px)
- Subtle shadow
- Hover effect (scale + shadow increase)
- Border with low opacity
```

### Кнопки
```css
Primary:   gradient blue, white text
Secondary: outline, hover fill
Size:      large (lg), padding generous
```

### Анимации
```css
- Fade in на появление (с delay)
- Hover scale (105-110%)
- Smooth transitions (300-500ms)
```
